Database Demo

Hi! This is just a quick demo of fetching and updating data to a remote database (mainly for myself and whenever I showcase this to someone else)

Notes:

  • Security features: none. This means there's no safety check to make sure the data being sent or gotten really should be. Like someone could spam update this and fill the storage of my server :<

Features

Currently, this demo supports:

  • Checking online status of the database
  • Fetching data with resource path
  • Setting data to a resource path
