Database Demo v2

This should be functionally similar to the other database, but faster. This is done with the methods listed below.

  • Websockets (instead of or in conjunction with HTTP requests)
  • A real database like SQL (instead of JSON files)
  • Caching

Features

Currently, this demo supports:

  • Checking online status of the database
  • Fetching data with resource path
  • Setting data to a resource path
